# Four-Router Lab Exam

## Summary

This folder contains the lab exam version of the four-router static routing activity in Cisco Packet Tracer.  
It uses the same ring topology (R1–R4), with two LANs per router and end-to-end connectivity through static routes.

## Files

| File | Description |
|------|-------------|
| `CAHAPAY_IT202WM_LABEXAM.pkt` | Packet Tracer lab exam project file |
| `Lab Exam Code Configuration` | CLI configuration reference for routers, static routes, and PC addressing |
| `your_demise_part_1.png` | Topology image/reference |
| `your_demise_part_1_v2.png` | Updated topology image/reference |

## How to Use

1. Open `CAHAPAY_IT202WM_LABEXAM.pkt` in Cisco Packet Tracer.
2. Apply or verify router/PC settings using `Lab Exam Code Configuration`.
3. Validate connectivity with `ping` between local and remote subnets.
