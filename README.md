# Networking Guide

A collection of Cisco Packet Tracer lab configurations covering fundamental networking concepts — from basic switch connectivity and VLANs to multi-router static routing. Each folder contains a Packet Tracer project file (`.pkt`), a CLI code guide, and a PDF summary.

## Labs

| Folder | Description |
|--------|-------------|
| [Basic Switch to Switch Connectivity and VLAN](./Basic%20Switch%20to%20Switch%20Connectivity%20and%20VLAN/) | Configure three switches and four PCs on a shared VLAN with management IP addresses |
| [Two-Router Network Configuration](./Two-Router%20Network%20Configuration/) | Connect two routers via a serial WAN link and route traffic between four subnets using static routes |
| [Four-Router Network Configuration](./Four-Router%20Network%20Configuration/) | Build a ring topology with four routers, eight subnets, and full-mesh static routing between all 16 PCs |

## Tools Used

- **Cisco Packet Tracer** — network simulation software for building and testing virtual lab topologies
- **Cisco IOS CLI** — command-line interface used to configure routers and switches
